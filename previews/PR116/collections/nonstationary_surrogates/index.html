Surrogates for nonstationary time series

Truncated Fourier surrogates

TFTS

Truncated Fourier transform surrogates (TFTS) preserve some portion of the frequency spectrum of the original signal. Here, we randomize the 95% highest frequencies, while keeping the 5% lowermost frequencies intact.

using TimeseriesSurrogates
n = 300
a = 0.7
A = 20
σ = 15
x = cumsum(randn(n)) .+ [(1 + a*i) .+ A*sin(2π/10*i) for i = 1:n] .+
    [A^2*sin(2π/2*i + π) for i = 1:n] .+ σ .* rand(n).^2;


fϵ = 0.05
s_tfts = surrogate(x, TFTS(fϵ))
surroplot(x, s_tfts)

One may also choose to preserve the opposite end of the frequency spectrum. Below, we randomize the 20% lowermost frequencies, while keeping the 80% highest frequencies intact.

using TimeseriesSurrogates
n = 300
a = 0.7
A = 20
σ = 15
x = cumsum(randn(n)) .+ [(1 + a*i) .+ A*sin(2π/10*i) for i = 1:n] .+
    [A^2*sin(2π/2*i + π) for i = 1:n] .+ σ .* rand(n).^2;

fϵ = -0.2
s_tfts = surrogate(x, TFTS(fϵ))
surroplot(x, s_tfts)

TAAFT

Truncated AAFT surrogates (TAAFT) are similar to TFTS surrogates, but adds the extra step of rescaling back to the original values of the signal, so that the original signal and the surrogates consists of the same values.

using TimeseriesSurrogates

# Example signal
n = 300; a = 0.7; A = 20; σ = 15
x = cumsum(randn(n)) .+ [(1 + a*i) .+ A*sin(2π/10*i) for i = 1:n] .+
    [A^2*sin(2π/2*i + π) for i = 1:n] .+ σ .* rand(n).^2;

# Preserve 5% of the power spectrum corresponding to the lowest frequencies
s_taaft_lo = surrogate(x, TAAFT(0.05))
surroplot(x, s_taaft_lo)
using TimeseriesSurrogates

# Example signal
n = 300; a = 0.7; A = 20; σ = 15
x = cumsum(randn(n)) .+ [(1 + a*i) .+ A*sin(2π/10*i) for i = 1:n] .+
    [A^2*sin(2π/2*i + π) for i = 1:n] .+ σ .* rand(n).^2;

# Preserve 20% of the power spectrum corresponding to the highest frequencies
s_taaft_hi = surrogate(x, TAAFT(-0.2))
surroplot(x, s_taaft_hi)

Truncated FT surrogates with trend removal/addition

For time series with strong trends, Lucio et al. (2012)[Lucio2012] proposes variants of the truncated Fourier-based surrogates wherein the trend is removed prior to surrogate generation, and then added to the surrogate again after it has been generated. This yields surrogates quite similar to those obtained when using truncated Fourier surrogates (e.g. TFTS), but reducing the effects of endpoint mismatch that affects regular truncated Fourier transform based surrogates.

In principle, any trend could be removed/added to the signal. For now, the only option is to remove a best-fit linear trend obtained by ordinary least squares regression.

TFTD

The TFTD surrogate is a random Fourier surrogate where the lowest frequencies are preserved during surrogate generation, and a linear trend is removed during preprosessing and added again after the surrogate has been generated. 

using TimeseriesSurrogates

# Example signal
n = 300; a = 0.7; A = 20; σ = 15
x = cumsum(randn(n)) .+ [(1 + a*i) .+ A*sin(2π/10*i) for i = 1:n] .+
    [A^2*sin(2π/2*i + π) for i = 1:n] .+ σ .* rand(n).^2;

s = surrogate(x, TFTDRandomFourier(true, 0.02))
surroplot(x, s)

TFTDAAFT

The TFTDAAFT[Lucio2012] are similar to TFTDRandomFourier surrogates, but also adds an additiona rescaling step, so that the time series has the same values as the original time series.

using TimeseriesSurrogates

# Example signal
n = 300; a = 0.7; A = 20; σ = 15
x = cumsum(randn(n)) .+ [(1 + a*i) .+ A*sin(2π/10*i) for i = 1:n] .+
    [A^2*sin(2π/2*i + π) for i = 1:n] .+ σ .* rand(n).^2;

# Keep 2 % of lowermost frequencies.
s = surrogate(x, TFTDAAFT(0.02))
surroplot(x, s)

TFTDIAAFT

The TFTDIAAFT[Lucio2012] are similar to TFTDAAFT surrogates, but uses an iterative process to better match the power spectra of the original signal and the surrogate (anlogous to how the IAAFT method improves upon the AAFT method).

using TimeseriesSurrogates

# Example signal
n = 300; a = 0.7; A = 20; σ = 15
x = cumsum(randn(n)) .+ [(1 + a*i) .+ A*sin(2π/10*i) for i = 1:n] .+
    [A^2*sin(2π/2*i + π) for i = 1:n] .+ σ .* rand(n).^2;

# Keep 5% of lowermost frequences
s = surrogate(x, TFTDIAAFT(0.05))
surroplot(x, s)
